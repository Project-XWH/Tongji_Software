Parts Browser To design a part is quite difficult and should take too much factors into consideration, so we collect data from iGEM parts and build a search engine in order to make it easier to search a part with certain function. In this page, you can key the name of parts or the function you needed into the search box. If you have any questions about the parameters, you can drag your mouse to the question mark following, and it will give you instructions about the parameters you should key in.
